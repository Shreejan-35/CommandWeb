CommandConsole

A lightweight, fast and secure shell for all.

CommandConsole is a fast, secure and lightweight shell which can automate tasks for you. It takes care of the user and has a user-freindly interface.
It is a cross-platform application which can be used in windows, linux and macOS. It is open-source and anybody can contribute in it.
Happy life with CommandConsole.
Our motivation and inspiration is Powershell and we are very much grateful to it.
